# Better MC Screenshots

Better MC Screenshots is a **Minecraft mod** that completely upgrades the screenshot system. It adds modern, convenient features that make capturing, browsing, and sharing screenshots easier than ever — all without leaving the game.

## Features

- **Instant in-game screenshot preview** – check your screenshot immediately after taking it.
- **Customizable screenshot sound** – adjust the sound effect to your liking.
- **Enhanced chat notifications** – with extra details and interactive options, like quickly opening the preview or the screenshot folder.
- **Built-in screenshot gallery** – browse all your screenshots without minimizing the game.
- **Copy screenshots directly from the game** – share instantly without hunting for files.
- **Quick access to saved screenshots** – no more searching through folders.

## Installation

Download the mod from [Modrinth](https://modrinth.com/mod/better-mc-screenshots).

## Development

To build the project using Gradle, run the following commands in the project root, depending on your Minecraft version:

```bash
./gradlew build            # All versions
./gradlew 1.21.11:build    # version 1.21.11
./gradlew 1.21.10:build    # version 1.21.10
./gradlew 1.21.8:build     # version 1.21.8
